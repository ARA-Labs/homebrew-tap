# ARA-Labs Homebrew Tap

Homebrew formulae for [ARA-Labs](https://github.com/ARA-Labs) tools.

## Install `ara`

```bash
brew install ARA-Labs/tap/ara
```

Or tap first, then install:

```bash
brew tap ARA-Labs/tap
brew install ara
```

`ara` is the command-line runtime for the ARA viewer — it validates and serves
[Agent-Native Research Artifacts](https://github.com/ARA-Labs/ara-cli). Pre-built
binaries are provided for:

- macOS Apple Silicon (`aarch64-apple-darwin`)
- Linux x86_64 (`x86_64-unknown-linux-gnu`)

On other platforms, install from source instead:

```bash
cargo install ara-cli
```

## How this tap is maintained

`Formula/ara.rb` is generated by [cargo-dist](https://axodotdev.github.io/cargo-dist)
during the [`ara-cli`](https://github.com/ARA-Labs/ara-cli) release build (on each
`vX.Y.Z` tag) and published here. It points at the release tarballs on
`ARA-Labs/ara-cli` and verifies each with a SHA-256 checksum. Do not hand-edit
the formula — update it by cutting a new `ara-cli` release.
